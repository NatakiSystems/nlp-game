🤖 Welcome to the Words-to-Numbers AI Playground!

What is this?

A fun, lightweight, interactive web game designed to teach non-technical learners how traditional Machine Learning algorithms process human language.

🕹️ Features:
Level 1: The Bag of Words Blend-Up – Type in your own tricky sentences to see exactly how text-scrambling limitations completely blind traditional models to sentence structure.

Level 2: The TF-IDF Superstar Detector – Scan three fictional mini-storybooks to watch mathematical weighting filter out common background noise words (like "the") and highlight unique concept words dynamically.

Built entirely in Python using Streamlit! 🎈
